# Etch-a-Sketch

Implement etch-a-sketch via JS and HTML and sprite of CSS

## Working
Sketch on Hover + Click, involves 2 events:
- `mousedown`
- `mouseover`

## Features
Sketch with :
- Choose by Color
- Rainbow
- GrayScale + Transparent
